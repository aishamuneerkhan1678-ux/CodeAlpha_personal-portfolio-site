# Personal Portfolio Website

## Overview

This is my personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, projects, education, resume, and contact information in a modern and responsive design.

The website is designed to provide visitors with a professional overview of my background, technical skills, and development projects.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern user interface with animations and hover effects
- Smooth scrolling navigation
- About Me section
- Skills section with animated progress bars
- Projects showcase
- Resume section with download option
- Contact section
- Interactive user experience using JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts
